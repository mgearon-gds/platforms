Page designer

View the prototype

A basic form page designer with the following features:

  • the interface and forms are accessible and mobile friendly
  • you create one page at a time, using a looping wizard
  • pages are designed via a form, rather than directly edited
  • there's a persistent mobile-sized preview pane on most pages
  • the preview pane is functional, so you can use it to navigate the form

Hypotheses to test:

  • a wizard is usable by non-DDaT professionals
  • a wizard usefully constrains the space of possible page designs
  • a persitant preview pane helps users navigate the form
  • a persitant preview pane helps users understand which elements go where
  • a persitant preview pane provides reassurance about the final form
